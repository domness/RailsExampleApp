# Example Rails App

This is an example Rails app with a number of things set up including:

- Rails
- SQLite3
- HAML
- Devise (with HAML views)
- Twitter Bootstrap
- JQuery
- RSpec
- Capybara
- SimpleCov
- User model (with devise)
- LESS/SASS (Up to you to decide)
- Coffeescript
- Jasmine (Javascript testing)
